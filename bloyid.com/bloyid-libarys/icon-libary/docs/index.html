Installation

Add the following line in the <head> section of your HTML to link the library:

<script src="path/to/bloyid-icons.js"></script>

Optionally, include your CSS file for custom styles:

<link rel="stylesheet" href="path/to/styles.css">

Usage

To use an icon, insert a <blicon> tag in your HTML:

<bIcon name="iconName" color="desiredColor" size="desiredSize" class="optionalClass"></bIcon>

Attributes:

  • name: (required) The name of the icon you want to display.
  • color: (optional) The color of the icon (default is currentColor).
  • size: (optional) The size of the icon (default is 50px).
  • class: (optional) Custom CSS class names for additional styling or animations.

Customization

You can customize the icons using CSS:

Custom Classes:

<bIcon name="icon1" class="my-custom-icon"></bIcon>

Then in your CSS:

.my-custom-icon {
    filter: drop-shadow(2px 2px 2px gray);
    transition: transform 0.3s ease;
}
.my-custom-icon:hover {
    transform: scale(1.1);
}

Animations:

.rotate {
    animation: rotation 2s infinite linear;
}

@keyframes rotation {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}
<bIcon name="icon1" class="rotate"></bIcon>
